# docx

Create, read, and edit Word documents (.docx). Covers document creation, text extraction, formatting, tables, images, headers/footers, and conversion.

## Usage

```
/docx create a professional report from these notes
/docx extract all tables from contract.docx
/docx convert this markdown to a Word document
```

## Dependencies

- `python-docx` — Python API for .docx creation and editing
- `pandoc` (system) — document conversion and content extraction
- `libreoffice` (system) — PDF conversion, .doc to .docx conversion
