# xlsx

Create, read, edit, and analyze Excel spreadsheets (.xlsx). Covers formulas, formatting, data analysis, charts, and financial model standards.

## Usage

```
/xlsx create a budget spreadsheet from these categories
/xlsx analyze the sales data in Q4_report.xlsx
/xlsx convert this CSV to a formatted Excel file
```

## Dependencies

- `openpyxl` — Excel creation/editing with full formatting and formula support
- `pandas` — Data analysis, CSV/Excel I/O
- `libreoffice` (system) — Formula recalculation
