# skill-creator

Guide for creating new Octo skills for the marketplace. Covers skill anatomy, progressive disclosure, creation process, and validation.

## Usage

```
/skill-creator
/skill-creator I want to create a skill for database migrations
```

## What It Does

1. Walks through understanding the skill's purpose and triggers
2. Plans reusable contents (scripts, references, assets)
3. Guides writing the SKILL.md with proper frontmatter
4. Validates against marketplace rules
5. Helps test and iterate
