# pptx

Create and edit PowerPoint presentations with strong design emphasis. Includes color palettes, typography guidelines, layout rules, and mandatory visual QA.

## Usage

```
/pptx create a pitch deck for our product launch
/pptx convert these notes into a presentation
/pptx update the quarterly report slides
```

## Dependencies

- `python-pptx` — Python API for PowerPoint creation/editing
- `Pillow` — Image processing
- `libreoffice` (system) — PDF conversion for visual QA
- `poppler` (system) — PDF to image conversion

## Design Features

- 7 pre-built color palettes
- Typography and spacing guidelines
- Anti-pattern list (what NOT to do)
- Mandatory visual QA process with fix-and-verify loop
